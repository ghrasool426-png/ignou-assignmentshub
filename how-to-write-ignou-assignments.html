How to Write IGNOU Assignments (2025)

Complete Step-by-Step Guide for All IGNOU Courses

Writing IGNOU assignments correctly is very important for scoring good marks. Many students lose marks not because of poor knowledge, but due to wrong format, poor presentation, and misunderstanding of questions.

In this guide, you will learn how to write IGNOU assignments step by step, including format, word limit, answer structure and common mistakes to avoid.

1. Understand the IGNOU Assignment Question

Before writing anything, read the question carefully. Identify key terms like discuss, explain, analyze, describe. Answer exactly what is asked, not what you know.

2. Follow Correct IGNOU Assignment Format

IGNOU follows a simple handwritten format unless specified otherwise. Always use clean handwriting and proper spacing.

  • Write on A4 size paper
  • Use blue or black pen only
  • Leave proper margins on all sides
  • Number pages clearly

3. Assignment Front Page Details

The front page must contain correct details such as:

  • Student Name
  • Enrollment Number
  • Programme Code (BA, MA, BSc, BBA, MSc etc.)
  • Course Code
  • Study Centre Code
  • Session (January / July)

4. Word Limit for IGNOU Assignments

IGNOU usually specifies word limits:

  • 10 marks question → 500 words
  • 20 marks question → 1000 words

Do not write too short or excessively long answers.

5. How to Structure Your Answers

A well-structured answer always gets better marks:

  • Introduction – Brief overview of the topic
  • Main Body – Explain points with headings
  • Conclusion – Summarize key ideas

6. Use Simple Language & Examples

Use clear and simple language. Avoid copying from books or the internet. Where possible, add examples, diagrams or case studies.

7. Common Mistakes to Avoid

  • Copy-paste answers
  • Ignoring word limit
  • Wrong course code on front page
  • Poor handwriting
  • Late submission

8. Important Submission Tips

Always submit assignments before the deadline at your study centre. Keep a photocopy or scanned copy for safety.
