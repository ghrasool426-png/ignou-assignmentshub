IGNOU Assignment Result

Check Assignment Marks, Status & Latest Updates

The IGNOU Assignment Result plays a crucial role in the final evaluation of students enrolled in IGNOU programmes. Since assignments carry significant weightage, it is important for learners to regularly check their assignment status and marks.

This guide explains how to check IGNOU assignment results online, possible reasons for delay, and what to do if marks are not updated. This page is created strictly for educational guidance.

IGNOU Assignment Result Overview

UniversityIndira Gandhi National Open University (IGNOU)
Result TypeAssignment Marks / Assignment Status
ModeOnline
Required DetailsEnrollment Number & Programme Code
Official Websitewww.ignou.ac.in

Why IGNOU Assignment Result Is Important

  • Assignment marks are added to the final course score
  • Passing assignments is mandatory to clear a subject
  • Good assignment marks improve overall grade
  • Final result is incomplete without assignment marks

How to Check IGNOU Assignment Result

  1. Visit the official IGNOU website
  2. Go to the Student Support section
  3. Click on Assignment Status / Result
  4. Enter your Enrollment Number
  5. Select your Programme Code
  6. Click on Submit

Details Shown in Assignment Result

  • Student Name
  • Enrollment Number
  • Programme Code
  • Course Code
  • Assignment Marks
  • Evaluation Status
  • Remarks (if any)

Why Assignment Marks May Not Be Showing

  • Assignment evaluation still in progress
  • Delay at study centre or regional centre
  • Late submission of assignment
  • Technical delay in uploading marks

What To Do If Marks Are Not Updated

  • Visit your study centre for confirmation
  • Contact the regional centre
  • Keep assignment submission receipt safe
  • Wait patiently as results are updated in batches

Frequently Asked Questions

Are assignments compulsory in IGNOU?

Yes, assignments are mandatory to pass any IGNOU course.

Can I appear in exam without assignment marks?

You may appear, but final result will not be declared without assignment marks.

How long does IGNOU take to update assignment marks?

It may take a few weeks to several months depending on evaluation load.
